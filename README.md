# Game List App

## Description

Game List App is a web application built using Express, EJS, and Bootstrap. This app allows users to view a list of games, including their platform and status. It also features a simple responsive design. Displays a list of games retrieved from a back-end API or database.

- Node.js
- Express
- EJS
- Bootstrap

## Environment Variables

To configure the application, create a .env file in the root directory of your project and include the following variables:

    POSTGRES_HOST=your_postgres_host
    POSTGRES_USER=your_postgres_user
    POSTGRES_PASSWORD=your_postgres_password
    POSTGRES_DATABASE=your_postgres_database_name
    POSTGRES_DBPORT=your_postgres_port
    PORT=your_application_port
    EMBED_TWITCH_CHANNEL=your_twitch_channel_name
    EMBED_TWITCH_PARENT=your_twitch_parent_hostname
